# Start Here

Use this page to decide what to read based on your immediate discussion goal.

## If I Only Have 10 Minutes

**Recommended pages**

- [Learning Index](client-pos-preparation/learning-index.md)
- [Talking Points](meeting-outputs/talking-points.md)

**Focus**

- Separate confirmed facts, working assumptions, and open questions.
- Keep the BA + PM + FDE lens in mind.

**Practice**

> “Let’s separate confirmed facts, working assumptions, and open questions before moving into recommendations.”

## If I Need A Project Alignment Discussion

**Recommended pages**

- [Project Alignment Questions](meeting-outputs/project-alignment-questions.md)
- [Client POS / Sales Platform Preparation](client-pos-preparation/index.md)

**Focus**

- Clarify priority, expected output, stakeholders, and next milestone.
- Ask what output would be most useful to the team.

**Practice**

> “What would be the most useful output before the next touchpoint: a question list, current-state summary, AI use case view, or BRD tracker draft?”

## If I Need To Understand Insurance POS

**Recommended pages**

- [Insurance POS Basics](insurance-pos-basics/pos-basics.md)
- [Product Configuration](insurance-pos-basics/product-configuration.md)

**Focus**

- Understand FNA, illustration, application capture, document upload, and submission.
- Watch how product rules affect quotes, NIGO, referrals, and POS configuration.

**Practice**

> “Does the current POS / sales platform scope cover quote, illustration, application capture, document upload, and submission, or only part of that journey?”

## If I Need To Discuss Sales Journey AI

**Recommended pages**

- [AI Opportunities in Insurance](sales-journey-ai/opportunities.md)
- [Sales Journey AI Talking Points](sales-journey-ai/talking-points.md)
- [Identification Playbook](sales-journey-ai/identification-playbook.md)

**Focus**

- Start with lower-risk, agent-facing AI.
- Do not treat product recommendation, suitability, or underwriting decisioning as quick wins.

**Practice**

> “I suggest starting with approved-source, agent-facing, human-reviewed use cases before discussing recommendation or underwriting decisioning.”

## If I Need BRD / Template Reformat Preparation

**Recommended pages**

- [BRD Reformat Playbook](product-brd-template-reformat/brd-reformat-playbook.md)
- [Product Configuration](insurance-pos-basics/product-configuration.md)

**Focus**

- Clarify the count unit for the batch of legacy templates/items.
- Separate documentation clean-up from system configuration and export mapping.

**Practice**

> “Before reformatting, we should confirm count unit, target export format, required fields, and review owner.”

## If I Need Meeting-Ready Talking Points

**Recommended pages**

- [Talking Points](meeting-outputs/talking-points.md)
- [Discovery Questions](meeting-outputs/discovery-questions.md)
