# Sales Journey AI

這一節幫讀者把 AI idea 放回保險銷售 journey：user、pain point、data、human control、metric 和 compliance consideration。

## How To Use This Page

- 準備討論 AI 前，先讀 [Sales Journey AI Notes](ai-notes.md)。
- 想知道「有啲咩位可以加 AI」，讀 [AI Opportunities in Insurance](opportunities.md)。
- 要準備 workshop / discovery，讀 [Identification Playbook](identification-playbook.md)。
- 會前想攞句子直接講，讀 [Talking Points](talking-points.md)。
- 如果要排序 use cases，跳到 [AI Use Case Prioritisation](../ai-use-case-prioritisation/index.md)。

## Key Reminder

先由低風險、agent-facing、approved-source、human-reviewed use cases 開始。Product recommendation、suitability assessment、underwriting decisioning 唔應該包裝成簡單 quick wins。
