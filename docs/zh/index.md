# Insurance AI Learning Library

這是一個可分享的 web-style learning library，用於保險 POS、Sales Journey AI、BRD reformatting 和 AI use case prioritisation 的學習與討論。

!!! note "Disclaimer"
    本網站僅作個人學習與討論用途，並非客戶建議、法律意見、合規意見或正式項目建議。

## How To Use This Site

- 如果要快速建立背景，先讀 **Client POS / Sales Platform Preparation** 和 **Insurance POS Basics**。
- 如果要準備 AI 討論，讀 **Sales Journey AI** 和 **AI Use Case Prioritisation**。
- 如果要準備 team / project lead / client 對話，直接看 **Meeting / Discussion Outputs**。

## Working Principle

本 library 用 **AI Transformation Pathfinder** 角度整理材料：用 BA、PM、FDE 三個 lens，把業務需求、交付節奏、系統可行性、資料限制和 AI governance 連起來。

討論 AI 時，應連到 user pain point、approved data source、human review、audit trail 同 compliance sign-off。
