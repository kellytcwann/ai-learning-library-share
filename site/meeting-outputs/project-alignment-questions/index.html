    Project Alignment Questions¶

    這頁幫項目成員同 project lead 對焦，確認下一步優先級、交付物和準備方向。

    Opening¶

    「我想用 BA、PM、FDE 三個 lens 準備 insurance transformation workstream。想同你 confirm 一下，下一步 focus 喺邊度會最有用？」

    Scope / Priority¶

    • 「下一步最應該產出咩？Question list、current-state summary、deck content、BRD tracker，定 meeting notes？」
    • 「而家 priority 係current POS / sales platform assessment、AI use cases、BRD reformat，定 product configuration？」
    • 「會主要 support 邊類 stakeholder？Business、IT、vendor、compliance、operations，定 sales？」
    • 「下次 internal / client meeting 前，最需要先補邊方面 context？」
    • 「有冇已知 hypothesis 或 leadership concern，需要先 keep in mind？」

    Current POS / Sales Platform¶

    • 「今次最重要嘅 sales journey steps 係邊幾個？」
    • 「Team 係咪已經知道某啲 POS pain points？」
    • 「有冇現成 materials 可以睇，例如 screenshots、process maps、issue logs、BRDs、NIGO data？」
    • 「Product rules 主要係 configure 喺 current POS / sales platform、downstream systems，定而家好多都係 manual process？」
    • 「如果要快速補 POS 背景，有邊幾個 terms 或流程應該先搞清楚？」

    AI Transformation¶

    • 「係要 propose new AI use cases，定 assess client 已有 ideas？」
    • 「AI scope 應該先 focus agent-facing、operations-facing、product-team-facing，定 customer-facing？」
    • 「有冇邊啲 AI topic 對 client / compliance 特別 sensitive，要避免講得太過？」
    • 「應該用邊個 governance lens 去睇？Privacy、suitability、auditability、model risk，定全部都要 cover？」
    • 「如果要 prioritise AI use cases，team 偏向睇 value、feasibility、risk、data readiness，定 integration dependency？」

    BRD / Template¶

    • 「A batch of legacy templates/items 具體係咩？Products、riders、documents、rules，定 export rows？」
    • 「BRD task 係純 reformat documents，定要 extract product rules、create tracker、map export fields？」
    • 「Reformatted requirements 之後邊個 validate？」
    • 「需唔需要保留 source traceability，例如 original wording、file name、section/page？」
    • 「如果 old template 有 missing / ambiguous rules，應該點 log 同 assign owner？」

    Closing¶

    「為咗確保時間用得準，下次 touchpoint 前你覺得交邊樣最有幫助：question list、current-state summary、AI use case view，定 BRD tracker draft？」
